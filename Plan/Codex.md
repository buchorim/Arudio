# Codex — Arudio

## What Is This
Arudio is a browser-based audio mixer and editor concept for arranging, adjusting, previewing, and eventually exporting polished audio projects from a professional-feeling timeline interface. The first goal is a faithful migration of the existing prototype UI into a lighter stack so the owner can judge whether the product surface feels right before deeper audio editing work begins.

## Who Is It For
Arudio is for independent creators, musicians, video editors, and podcast editors who want a browser-accessible editor with familiar DAW-style controls, multitrack layout, clip controls, meters, and an approachable workflow.

## How to Use
1. Open the Arudio editor in the browser.
2. Review or create a project from the left sidebar.
3. Use the timeline to select clips, move the playhead, and adjust clip placement.
4. Use the right inspector to adjust clip gain, pan, pitch, reverse state, and effect toggles.
5. Use the transport controls to play, stop, and review the session.
6. Use export once real file-backed rendering is implemented and validated.

## Key Constraints & Assumptions
The initial implementation must prioritize visual parity with the current prototype over full production audio functionality. Browser-native audio APIs are assumed to be the core technology path. Exact parity with desktop DAWs, native plugin support, hardware driver routing, and full mastering workflows are out of scope until later phases define them explicitly.
